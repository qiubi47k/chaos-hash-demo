# ChaosHash - 混沌加密内核演示

基于布尔拓扑自演化系统产出的高阶有理递推结构，构建的私有非线性哈希方案。

## 原理

系统在无约束布尔拓扑空间中自主演化，产出9-28阶有理分式系数线性递推结构。每条递推天然具备振荡发散特性——微小输入扰动会在递推过程中被放大数百倍，产生不可预测的混沌输出。

这正是密码学雪崩效应的核心需求。

## 特点

- **379套独立变换内核**：每条递推结构不同、系数不同、动力学行为不同
- **P分裂结构**：发散速度平缓可控，计算不溢出不崩溃
- **有理分式系数**：严格数学结构，非浮点近似
- **私有方案**：非公开标准算法，攻击者无法获取具体实现

## 在线体验

直接打开 [chaos_hash_demo.html](https://qiubi47k.github.io/chaos-hash-demo/chaos_hash_demo.html) 即可体验。

输入任意文本，切换不同内核，观察：
1. 输出哈希完全不同
2. 改动1个字符，输出面目全非（雪崩效应）
3. 底部动力学曲线展示递推的混沌振荡过程

## 技术细节

- 初始状态：输入文本字符编码归一化为数值序列
- 递推计算：使用多项式系数做50步线性递推
- 哈希输出：取最终状态值转256-bit十六进制
- 所有计算在浏览器端完成，无需服务端

## 背景

本项目是布尔拓扑自演化系统的副产品。系统在无人类规则约束的条件下，自发收敛出具有稳定统计分布的高阶递推结构。本demo从中选取了4条代表性内核用于演示。
